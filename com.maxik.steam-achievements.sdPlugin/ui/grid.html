How to use the grid

Grid Cell: Place multiple cells on your Stream Deck to create a gallery of achievement icons. Each cell shows one achievement with a rarity-colored border.

Slot Index: Leave as "Auto" for automatic ordering, or set a specific number (0-based) to control which grid position this cell occupies.

Navigation: Add "Grid: Previous Page" and "Grid: Next Page" actions to paginate through all achievements.

Info & Sort: Add "Grid: Info & Sort" to see progress stats. Press it to cycle through sort modes: Default, Rarest First, Alphabetical, Locked Only, Unlocked Only.

Color legend: Gray border = Common (>50%), Green = Uncommon (20-50%), Blue = Rare (5-20%), Purple = Ultra Rare (1-5%), Gold = Legendary (<1%).
