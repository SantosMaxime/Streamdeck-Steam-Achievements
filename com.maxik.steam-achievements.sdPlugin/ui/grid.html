How to use the grid

Grid Cell: Place multiple cells on your Stream Deck. In achievements mode each cell shows one achievement; in games mode each cell shows a game tile.

Sort order: Changes the order achievements are displayed. Applies immediately.

Slot Index: Leave as "Auto" for automatic ordering, or set a specific number (0-based) to control which grid position this cell occupies.

Navigation: Add "Grid: Previous Page" and "Grid: Next Page" actions to paginate.

Grid Info: Shows progress stats in achievements mode. In games mode, press it to go back to achievements.

Color legend: Gray = Common (>50%), Green = Uncommon (20-50%), Blue = Rare (5-20%), Purple = Ultra Rare (1-5%), Gold = Legendary (<1%).
