How it works

Press the key to load the currently running game's achievements. Or pick a game above to load it immediately.

Load my games: Fetches your Steam library. Click any game to load its achievements into the grid right away.

Game App ID: Leave empty to auto-detect the running game, or type a specific App ID (the number in store.steampowered.com/app/440/).

The plugin automatically switches to the correct grid profile for your device.
