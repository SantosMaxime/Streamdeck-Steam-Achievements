Press "Pick a game to pin" to fetch your library.
How it works

Browse library on grid: Switches the grid to games mode — browse all your games as tiles. Click a tile to load its achievements.

Press the key: Loads the currently running game's achievements (or the pinned game if set). If no game is running, opens the games browser.

Game App ID: Pin a specific game (the number in store.steampowered.com/app/440/). Leave empty for auto-detect.
