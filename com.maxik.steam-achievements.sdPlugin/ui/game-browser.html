How it works

Press the key to load a game's achievements into the grid and automatically switch to the grid profile.

Pick a game: Use "Load my games" to browse your Steam library and pick a game. The App ID field is set automatically when you click a game.

Game App ID: Leave empty to detect the currently running game. Or set a specific App ID to always load that game. Find IDs on store.steampowered.com (the number in /app/440/).

Auto profile: The plugin ships with bundled grid profiles for all Stream Deck models — Standard, Mini, XL, +, and Neo. The correct one is selected automatically based on your device. No setup needed.

Profile override: If you want to use your own custom profile instead of the bundled one, enter its exact name here (as defined in your Stream Deck software).
