How to use

1. Get your Steam Web API key from the link above.

2. Enter your 64-bit Steam ID (use steamid.io to find it).

3. These credentials are shared across all Steam Achievement Hunter actions — configure once here, and every action is ready to use.

4. Press the Settings key on your Stream Deck to test the connection. The key shows Configured ✓ when credentials are present, or Error if the test fails.
