How to use

1. Fill in your Steam API key and 64-bit Steam ID above (shared with all actions).

2. Set the polling interval (how often the radar checks for updates).

3. Choose what happens when you press the key: open a YouTube or Steam community guide for the displayed achievement.

4. When an achievement is unlocked, a 10-second celebration animation plays automatically.
